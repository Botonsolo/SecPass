# Changelog - v3.1.0 (SPA & Dark Redesign)

## [3.1.0] - 2026-02-17

### Added
- **Single Page Application (SPA)**: Security modules (Audit, Breach, Cracking, Reuse) now load dynamically within the dashboard container without page reloads.
- **Security Logic Documentation**: Added `security_logic_explanation.md` detailing the inner workings of the security features.
- **SPA Navigation Logic**: New `showSecurityModule` and `hideSecurityModule` functions in `app.js`.

### Changed
- **Thematic Redesign**: Implemented a "Premium Dark Glassmorphism" theme matching the login page.
- **UI/UX Improvements**: Smoother transitions, improved progress bars, and high-contrast status badges.
- **Version Update**: Application version bumped to `v3.1.0 PRO`.

### Fixed
- Improved sidebar contrast and interactive feedback.
- Fixed layout shifts during dynamic content loading.
