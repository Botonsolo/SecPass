# Instrucciones para v2.1.4

Para ver los cambios (diseño compacto, layout horizontal y correcciones), DEBES ejecutar la aplicación desde ESTA carpeta.

## Pasos para ejecutar:
1. Asegúrate de que no haya otros procesos de Python corriendo.
2. Abre una terminal en este directorio:
   `cd "c:\Users\jorge\Desktop\CIBER\Puesta en Produccion\Proyecto 001 - Gestor de Contraseñas\versions\v2.1.4_compact_redesign"`
3. Ejecuta el comando:
   `python app.py`
4. Abre [http://127.0.0.1:5000](http://127.0.0.1:5000) en tu navegador.
5. **MUY IMPORTANTE**: Pulsa `Ctrl + F5` en el navegador para forzar la actualización de los archivos CSS y JS.

Si sigues viendo la versión v2.1.3 o el diseño antiguo, es porque tu navegador tiene guardada la versión anterior en caché. El refresco forzado (Ctrl+F5) lo solucionará.
